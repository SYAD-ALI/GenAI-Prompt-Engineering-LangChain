## 📋 Tasks Implemented

### ✅ Task 1: Replace Hardcoded Prompts

Converted static f-string prompts into reusable LangChain `PromptTemplate`.

### ✅ Task 2: Multi-Input Prompt System

Created a prompt template that accepts multiple inputs:

* topic
* audience
* tone

---

### ✅ Task 3: Prompt Variations Engine

Developed multiple prompt styles:

* Teaching
* Interview
* Storytelling

---

### ✅ Task 4: ChatPromptTemplate System

Implemented role-based prompting using:

* System role
* User role
  Roles used:
* Teacher
* Interviewer
* Motivator

---

### ✅ Task 5: Input Validation Layer

Added validation rules for:

* Audience → beginner, intermediate, expert
* Tone → formal, casual, fun

Invalid inputs are handled using default values or errors.

---

### ✅ Task 6: Prompt Generator App

Created a function to generate prompts dynamically using:

* topic
* audience
* tone
* style

---

### ✅ Task 7: Template Reusability Test

Tested one template with multiple inputs to ensure:

* Same structure
* Different outputs
